# TranskribusPyClient

A Pythonic API and some command line tools to access the Transkribus server via its REST API


**Commands**

* do_addDocToCollec.py
* do_createCollec.py
* do_deleteCollec.py
* do_deleteJob.py
* do_duplicateDoc.py
* do_getJobStatus.py
* do_listCollec.py
* do_listPageLocks.py

* do_analyzeLayout.py
* do_htrHmm.py
* do_htrRnn.py
* do_listHtrHmm.py
* do_listHtrRnn.py

* do_login.py
* do_logout.py

* Transkribus_downloader.py
* TranskribusDU_transcriptUploader.py
